    让我们一步一步地了解 strcmp、strlen 和 strcat 这三个标准C库函数的用法。

    1. strcmp 用法¶

    strcmp 用于比较两个字符串的字典序。它有以下的函数原型：

    Copyint strcmp(const char *str1, const char *str2);
    • 返回值

    ：

    • 0：如果两个字符串相等。
    • <0：如果 str1 小于 str2。
    • >0：如果 str1 大于 str2。

    示例代码：

    Copy#include <stdio.h>
#include <string.h>

int main() {
    char str1[] = "hello";
    char str2[] = "world";

    int result = strcmp(str1, str2);

    if (result < 0) {
        printf("str1 is less than str2
");
    } else if (result == 0) {
        printf("str1 is equal to str2
");
    } else {
        printf("str1 is greater than str2
");
    }

    return 0;
}

    2. strlen 用法¶

    strlen 用于计算字符串的长度（不包括末尾的空字符 \0）。它有以下的函数原型：

    Copysize_t strlen(const char *str);
    • 返回值：字符串的长度。

    示例代码：

    Copy#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "hello";

    size_t length = strlen(str);

    printf("The length of the string is: %zu
", length);

    return 0;
}

    3. strcat 用法¶

    strcat 用于将两个字符串连接起来。它有以下的函数原型：

    Copychar *strcat(char *dest, const char *src);
    • 返回值：指向结果字符串的指针。

    注意：目标字符串 dest 必须有足够的空间来存放连接后的结果，否则会导致缓冲区溢出。

    示例代码：

    Copy#include <stdio.h>
#include <string.h>

int main() {
    char dest[20] = "hello";  // 需要足够的空间存放连接后的字符串
    char src[] = " world";

    strcat(dest, src);

    printf("The concatenated string is: %s
", dest);

    return 0;
}

    结合使用 strcmp、strlen strcpy 和 strcat 的示例¶

    让我们编写一个示例程序来展示如何结合使用这些函数：

    Copy#include <stdio.h>
#include <string.h>

int main() {
    char str1[50] = "hello";
    char str2[] = "world";
    char str3[] = "hello world";

    // 使用 strcmp 比较字符串
    int cmp_result = strcmp(str1, str2);
    if (cmp_result < 0) {
        printf("str1 is less than str2
");
    } else if (cmp_result == 0) {
        printf("str1 is equal to str2
");
    } else {
        printf("str1 is greater than str2
");
    }

    // 使用 strlen 计算字符串长度
    size_t length1 = strlen(str1);
    size_t length2 = strlen(str2);
    printf("Length of str1: %zu
", length1);
    printf("Length of str2: %zu
", length2);

    // 使用 strcat 连接字符串
    strcat(str1, " ");
    strcat(str1, str2);
    printf("Concatenated string: %s
", str1);

    // 比较连接后的字符串与预期结果
    if (strcmp(str1, str3) == 0) {
        printf("The concatenated string is correct.
");
    } else {
        printf("The concatenated string is incorrect.
");
    }

    return 0;
}

    运行上述程序将会得到如下输出：

    Copystr1 is less than str2
Length of str1: 5
Length of str2: 5
Concatenated string: hello world
The concatenated string is correct.

    通过以上示例，你可以了解如何使用 strcmp、strlen 和 strcat 进行字符串比较、长度计算和连接操作。

    strcpy 是一个用于复制字符串的标准库函数。它的原型是：

    Copychar *strcpy(char *dest, const char *src);
    • 参数：
    • dest：目标字符串指针，必须指向足够大的空间以容纳源字符串及其终止的空字符。
    • src：源字符串指针。
    • 返回值：返回目标字符串指针 dest。

    示例代码¶

    下面是一个使用 strcpy 的简单示例：

    Copy#include <stdio.h>
#include <string.h>

int main() {
    char src[] = "Hello, World!";
    char dest[50]; // 确保 dest 足够大以存放 src 的内容

    // 使用 strcpy 将 src 复制到 dest
    strcpy(dest, src);

    // 打印目标字符串
    printf("Source: %s
", src);
    printf("Destination: %s
", dest);

    return 0;
}

    解释¶

    1. char src[] = "Hello, World!";：声明并初始化源字符串 src。
    2. char dest[50];：声明目标字符串 dest，确保它有足够的空间存放源字符串及其终止的空字符。
    3. strcpy(dest, src);：将 src 的内容复制到 dest。
    4. printf：打印源字符串和目标字符串以验证复制是否成功。

    运行这段代码将输出：

    CopySource: Hello, World!
Destination: Hello, World!

{
